# Semantic Search Engine

Semantic Search Engine is a simple NLP project that retrieves documents based on their meaning rather than exact keyword matching. It uses transformer-based embeddings and cosine similarity to find the most relevant document for a user's query.

## Features

* Semantic search using embeddings
* Cosine similarity-based retrieval
* Natural language query support
* Beginner-friendly implementation

## Tech Stack

* Python
* Sentence Transformers
* Scikit-learn
* NumPy

## Run

```bash
pip install sentence-transformers scikit-learn numpy

python app.py
```
